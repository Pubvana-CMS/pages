# Pubvana Pages

Static pages module for Pubvana CMS. Provides CRUD for standalone pages with slug-based routing.

## Requirements

- PHP 8.1+
- [Flight School](https://github.com/enlivenapp/flight-school) ^0.2
- [Flight Shield](https://github.com/enlivenapp/flight-shield) ^0.1
- [Migrations](https://github.com/enlivenapp/migrations) ^0.1
- [Active Record](https://github.com/flightphp/active-record) ^0.7

## Installation

```bash
composer require pubvana/pages
```
